Este repositório detém do objetivo de centralizar os documentos gerados durante a confecção do trabalho e também proporcionar um detalhamento sobre um material extra que foi desenvolvido.
Durante o desenvolvimento do projeto foi desenvolvido duas APIs que são:
* [Data Calculate Shipping](https://github.com/EdiPSilva/data-calculate-freigh)
    * *Responsável por gerir o cadastrameto dos dados*
* [Calculate Shipping](https://github.com/EdiPSilva/calculate-freight)
    * *Responsável por realizar o cálculo de frete*
